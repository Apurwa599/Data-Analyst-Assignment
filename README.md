## Data-Analyst-Assignment- EdTech Data Analytics Case Study
### Overview
This project analyzes multiple datasets from an EdTech platform to understand student acquisition, engagement, pricing strategies, and satisfaction levels.
The analysis includes funnel conversion analysis, webinar performance evaluation, pricing A/B testing, NPS analysis, and LMS engagement trends to identify key growth opportunities and improve learning outcomes.

### Dataset A – Funnel Analysis
Analyzed the student acquisition funnel to understand conversion across stages such as:

Visits, Leads, Qualified Leads, Enrollments

Goal:
Identify where the largest drop-offs occur in the funnel and suggest improvements to increase conversion.

### Dataset B – Webinar Analysis
Examined webinar performance and engagement.
Key focus areas:

Attendance rate, Engagement level, Conversion impact

Objective:
Understand whether webinars effectively convert leads into enrolled students.

### Dataset C – Pricing A/B Test
Performed statistical analysis to compare Variant A vs Variant B pricing strategy.
Hypothesis Testing:

Null Hypothesis (H0): Both variants perform equally.

Alternative Hypothesis (H1): Variant B improves conversion.

Result:
Since p-value < 0.05, we reject the null hypothesis.

Conclusion:
Variant B significantly improves conversion rate compared to Variant A.
